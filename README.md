# idv_multi_launcher

第五人格多开启动器

## 功能

- 自动从注册表获取游戏安装路径 (当获取不到的时候请把启动器放到第五人格的游戏目录里)
- 启动时自动解除多开限制
- 支持命令行指定启动实例数

## 用法

```
idv_multi_launcher.exe [数量]
```

- `idv_multi_launcher.exe` — 启动 1 个实例
- `idv_multi_launcher.exe 3` — 启动 3 个实例
